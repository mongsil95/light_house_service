# 배민 UX 라이팅 가이드

> 배달의민족(baemin.com)과 배민외식업광장(ceo.baemin.com)의 UX 라이팅 스타일을 분석하여 반려해변 서비스에 적용할 수 있는 가이드

## 📌 핵심 원칙

### 1. 친근하고 따뜻한 톤앤매너

- **존댓말과 반말의 균형**: 설명은 존댓말, 행동 유도는 친근한 반말
- **공감하는 말투**: 사용자의 입장에서 생각하고 말하기
- **격식 없는 자연스러움**: 딱딱한 공지문이 아닌 대화하는 느낌

### 2. 짧고 명확한 메시지

- 한 문장은 15자 내외로 간결하게
- 핵심만 전달하되, 따뜻함은 유지
- 불필요한 수식어 제거

### 3. 감각적이고 생동감 있는 표현

- 의성어, 의태어 적극 활용
- 시각적으로 상상할 수 있는 구체적 표현
- 리듬감 있는 문장 구조

## 📝 제목(Main Title) 작성법

### 배민 스타일 분석

| 배민 예시                    | 특징                           |
| ---------------------------- | ------------------------------ |
| "세상 모든 것이 식지 않도록" | 은유적, 감성적, 미션 지향      |
| "치킨, 지금 배민됩니다"      | 간결, 행동 유도, 브랜드 동사화 |
| "마음을 배달해드려요"        | 감정 전달, 서비스 본질 표현    |

### 반려해변 적용 예시

**기존:** 반려해변 입양 및 관리 시스템
**개선:** 우리 바다, 깨끗하게 지켜드립니다

**기존:** 해양 쓰레기 정화 활동
**개선:** 깨끗한 파도, 오늘도 함께 만듭니다

**기존:** 전문가 Q&A
**개선:** 궁금한 건 물어보세요, 답은 여기 있어요

### 제목 작성 공식

```
[감정/가치] + [행동/결과] + [따뜻한 마무리]

예시:
- 바다를 지키는 + 가장 쉬운 방법 + 여기 있어요
- 혼자서도 + 함께여도 + 모두 환영합니다
- 작은 행동이 + 큰 변화를 + 만듭니다
```

## 🏷️ 소제목(Sub Title) 작성법

### 배민 스타일 분석

| 배민 예시                            | 특징                   |
| ------------------------------------ | ---------------------- |
| "시켜도 시켜도 무료배달"             | 반복을 통한 강조       |
| "365일 연중무휴 초신속 배달"         | 숫자 + 속도감          |
| "입점 가게 수 1위, 음식 리뷰 수 1위" | 구체적 수치, 신뢰 구축 |

### 반려해변 적용 예시

**카테고리 이름**

- 기존: 입양 안내 → **개선:** 시작이 궁금하신가요?
- 기존: 활동 가이드 → **개선:** 어떻게 시작할까요?
- 기존: 보고서/자료 → **개선:** 함께한 이야기들

**기능 설명**

- 기존: 해변 검색 → **개선:** 내 주변 해변 찾기
- 기존: 문의하기 → **개선:** 궁금한 게 있으세요?
- 기존: 활동 신청 → **개선:** 지금 바로 함께해요

### 소제목 작성 팁

1. **질문형으로 시작**: "~하신가요?", "~할까요?"
2. **행동 유도형**: "~해요", "~해보세요"
3. **공감형**: "~하시죠?", "~잖아요"

## 💬 본문(Body Text) 문체

### 배민 스타일 특징

**1. 리듬감 있는 문장**

```
배민: "필요한 건 무엇이든 한 시간 안에 배달합니다"
→ 짧은 호흡, 명확한 약속

반려해변: "궁금한 건 무엇이든 여기서 바로 찾아보세요"
```

**2. 반복과 대구를 통한 강조**

```
배민: "1인분도, 먼 거리도 무료배달"
→ ~도, ~도 구조로 포괄성 강조

반려해변: "처음이어도, 혼자여도 걱정 없어요"
```

**3. 친근한 종결어미**

```
배민 종결어미:
- ~해요 (친근함)
- ~니까 (이유 설명)
- ~드립니다 (정중함)
- ~하세요 (권유)

반려해변 적용:
- "바다를 지키는 첫걸음, 함께 시작해요"
- "어려울 것 같지만 생각보다 쉬우니까"
- "필요한 자료는 모두 제공해드립니다"
- "궁금한 점은 언제든 물어보세요"
```

## 🎯 CTA(Call To Action) 버튼 텍스트

### 배민 스타일

| 기능   | 일반적 표현 | 배민 스타일   |
| ------ | ----------- | ------------- |
| 가입   | 회원가입    | 지금 시작하기 |
| 신청   | 신청하기    | 바로 참여하기 |
| 문의   | 문의하기    | 물어보기      |
| 더보기 | 더보기      | 더 알아보기   |

### 반려해변 적용

```
❌ 피해야 할 표현          ✅ 권장 표현
------------------------------------------
입양 신청하기            → 함께하기
자료 다운로드            → 받아보기
활동 보고서 제출          → 이야기 남기기
문의 등록               → 질문하기
전문가 답변 보기          → 답변 확인하기
해변 검색               → 해변 찾기
```

## 🎨 카테고리별 문구 예시

### 1. 입양/시작 단계

**제목:** "바다와 친구가 되는 첫걸음"
**소제목:** "어렵지 않아요, 지금 바로 시작하세요"
**설명:** "우리 동네 해변을 선택하고, 간단한 정보만 입력하면 끝!
혼자서도, 친구들과 함께해도 좋아요."

### 2. 활동/참여 단계

**제목:** "오늘도 깨끗한 바다를 만듭니다"
**소제목:** "작은 손길이 모여 큰 변화를 만들어요"
**설명:** "쓰레기 줍기부터 캠페인까지,
할 수 있는 만큼만 하면 돼요. 부담 없이 참여하세요."

### 3. 커뮤니티/공유 단계

**제목:** "함께여서 더 힘이 나요"
**소제목:** "우리의 이야기를 나눠보세요"
**설명:** "같은 고민을 하는 동료들과 이야기 나누고,
전문가의 조언도 받아보세요. 혼자가 아니에요."

### 4. 문의/도움 단계

**제목:** "궁금한 건 물어보세요"
**소제목:** "등대지기가 친절하게 알려드릴게요"
**설명:** "활동 방법부터 지원 제도까지,
무엇이든 물어보세요. 빠르게 답변해드립니다."

## ✏️ 라이팅 체크리스트

글을 작성한 후 아래 항목을 확인하세요:

- [ ] **간결성**: 한 문장이 20자를 넘지 않나요?
- [ ] **친근함**: 딱딱하지 않고 대화하듯 편안한가요?
- [ ] **명확성**: 무엇을 해야 하는지 바로 이해되나요?
- [ ] **공감성**: 사용자의 상황과 감정을 이해하고 있나요?
- [ ] **행동 유도**: 다음 단계로 자연스럽게 이어지나요?
- [ ] **일관성**: 전체적인 톤앤매너가 일치하나요?
- [ ] **감성**: 이성적 정보뿐 아니라 감정도 전달되나요?

```

서브타이틀: 작은 손길이 모여 깨끗한 바다를 만듭니다
CTA: 지금 바로 시작하기
```

### 입양 안내 섹션

```
타이틀: 어렵지 않아요, 3분이면 충분해요
포인트 1: 가까운 해변 선택하기
포인트 2: 간단한 정보 입력하기
포인트 3: 활동 시작하기
CTA: 우리 해변 찾아보기
```

### 활동 가이드 섹션

```
타이틀: 할 수 있는 만큼만 하면 돼요
설명: 월 1회 방문도, 연 1회 정화도 모두 환영합니다.
혼자서도 좋고, 함께여도 좋아요.
CTA: 활동 방법 알아보기
```

### Q&A 섹션

```
타이틀: 궁금한 건 언제든 물어보세요
설명: 등대지기가 친절하게 답변해드립니다.
빠른 시간 안에 확인하고 알려드릴게요.
CTA: 질문하기
```

## 🌊 반려해변 전용 키워드 사전

### 핵심 가치 표현

- **지속가능성**: "오늘도, 내일도, 계속"
- **참여**: "함께", "같이", "우리"
- **쉬움**: "간단하게", "쉽게", "부담 없이"
- **신뢰**: "안심하고", "믿고", "책임지고"

### 감정 표현

- **환영**: "언제든", "모두", "누구나"
- **응원**: "힘내요", "할 수 있어요", "잘하고 있어요"
- **감사**: "고마워요", "덕분에", "함께해주셔서"

### 행동 유도

- **시작**: "시작해요", "첫걸음", "지금 바로"
- **참여**: "함께해요", "동참하기", "참여하기"
- **공유**: "나눠요", "들려주세요", "이야기해요"

## 💡 마무리 팁

1. **사용자 입장에서 생각하기**:
   - "우리가 하는 일"보다 "당신이 얻는 것"에 초점

2. **감정과 정보의 균형**:
   - 너무 감성적이면 신뢰 ↓
   - 너무 정보적이면 몰입 ↓

3. **일관된 캐릭터 유지**:
   - 친근하되 가벼워 보이지 않게
   - 전문적이되 어렵지 않게

4. **테스트와 개선**:
   - 실제 사용자에게 읽어보기
   - 소리 내어 읽었을 때 자연스러운지 확인
