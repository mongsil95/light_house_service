1. Framework & Language

Next.js
• React 기반 풀스택 프레임워크
• App Router 사용 권장
• SEO 및 성능 최적화에 유리
• 서버 컴포넌트 및 API Route 지원

TypeScript
• 정적 타입을 통한 안정성 확보
• 협업 시 커뮤니케이션 비용 감소
• 런타임 오류 사전 방지

npx create-next-app@latest --typescript

4. 프로젝트 폴더 구조 (권장)

app/
├─ layout.tsx # 공통 레이아웃
├─ page.tsx # 메인 페이지
├─ globals.css # 전역 스타일
├─ components/ # 공용 UI 컴포넌트
├─ hooks/ # 커스텀 훅
├─ lib/ # API, 유틸 함수
├─ types/ # 공통 타입 정의
└─ styles/ # 스타일 관련 파일
