## 1. 사이트 개요

### 1.1 사이트 목적

- **타겟 사용자**: 배달의민족을 이용하는 자영업자/점주
- **핵심 가치**: 외식업 사장님들의 비즈니스 성장 지원 플랫폼
- **포지셔닝**: 단순 주문 플랫폼을 넘어 종합 비즈니스 파트너

### 1.2 핵심 메시지

- "배민파트너와 함께 성장하는 외식업 생태계"
- 교육, 정보, 커뮤니티, 지원을 통한 전방위적 지원
- 파트너 중심의 쌍방향 소통 강조

## 3. UI/UX 디자인 분석

### 3.1 디자인 철학

#### 색상 시스템

- **Primary Color**: 배민 시그니처 청록색 계열
- **Secondary Color**: 따뜻한 오렌지/핑크 계열 (CTA, 강조)
- **Background**: 화이트 베이스 + 라이트 그레이 섹션 구분
- **Text**: 다크 그레이 (가독성 우선)

#### 타이포그래피

- **헤드라인**: 굵고 임팩트 있는 폰트 (사장님 친근감)
- **본문**: 중고딕 계열의 깔끔한 가독성
- **강조**: 컬러풀한 하이라이트, 이모지 활용

### 3.2 레이아웃 패턴

#### 섹션 구조

```
┌─────────────────────────────────────┐
│ Hero Section (주요 공지/CTA)         │
├─────────────────────────────────────┤
│ 최신 답변 (전문가 Q&A)               │
├─────────────────────────────────────┤
│ 인기 질문 (실시간 기준)              │
├─────────────────────────────────────┤
│ 참여광장 (퀴즈/투표)                 │
├─────────────────────────────────────┤
│ 외식업광장 숏폼 (비디오 콘텐츠)      │
├─────────────────────────────────────┤
│ 바로가기 (툴/계산기)                 │
├─────────────────────────────────────┤
│ 창업 프로모션 브랜드                 │
├─────────────────────────────────────┤
│ 배민파트너어워즈                     │
├─────────────────────────────────────┤

├─────────────────────────────────────┤

├─────────────────────────────────────┤
│ 장사노하우                           │
├─────────────────────────────────────┤
│ 추천 상품 (배민상회)                 │

│ 배민아카데미 무료 교육               │
├─────────────────────────────────────┤

├─────────────────────────────────────┤
│ 전문가Q&A                            │
└─────────────────────────────────────┘
```

#### 카드 디자인 특징

- **그리드 레이아웃**: 3~4열 카드 그리드
- **이미지 중심**: 썸네일 + 간결한 텍스트
- **호버 효과**: 부드러운 트랜지션
- **상태 표시**: "마감 임박", "NEW", "인기" 등 배지

### 3.3 UX 설계 원칙

#### 1. 즉각적 가치 제공

- 로그인 없이 대부분의 콘텐츠 열람 가능
- 첫 화면에서 핵심 정보/도구 바로 노출

#### 2. 스캔 가능성 (Scannability)

- 명확한 섹션 구분
- 아이콘과 이모지로 빠른 인지
- 숫자, 날짜로 시의성 강조

#### 3. 행동 유도 (Call-to-Action)

- "입점하기", "로그인", "참여하기" 등 명확한 CTA
- 고대비 버튼으로 시선 집중
- 혜택 금액을 숫자로 명시 (예: "총 63,000원 쿠폰팩")

## 4. 주요 기능 및 특징

### 4.1 핵심 기능

#### 전문가 Q&A 시스템

- **최신 답변 강조**: 실시간성 부각
- **전문가 프로필 표시**: 국립농산물품질관리원 등 신뢰성
- **카테고리 분류**: 세무·기타, 운영 등
- **인기순 정렬**: 실시간 기준 TOP 5

#### 참여형 콘텐츠

- **퀴즈**: 교육 + 참여 유도
  - 마감 기한 표시
  - 참여자 수 실시간 표시
  - 선택지 기반 인터랙션
- **투표/설문**: 커뮤니티 의견 수렴

#### 멀티미디어 콘텐츠

- **숏폼 비디오**: 1~2분 길이의 가이드
  - 썸네일 + 재생 시간
  - 주제: "배달팁 설정 방법", "부가세 신고 자료"
- **인포그래픽**: 복잡한 정보의 시각화

#### 유틸리티 도구

- 주휴수당 계산기
- 퇴직금 계산기
- 장사 캘린더
- 부가세 신고 자료 조회

### 4.2 커머스 통합

- **배민상회**: 배달용품 구매
  - 쿠폰 할인 강조
  - 할인율 표시 (예: 25%, 51%)
  - "직배송", "초특가" 등 라벨링

### 4.3 교육 플랫폼

- **배민아카데미**
  - 무료 온라인 교육
  - 영상 교육 + 오프라인 공간 지원
  - 조회수 표시로 인기 과정 확인 가능

## 5. 콘텐츠 전략

### 5.1 콘텐츠 유형

#### 1. 정보성 콘텐츠

- **가이드/매뉴얼**: "배달팁 설정 방법"
- **트렌드 리포트**: "2026 외식업 트렌드"
- **정책 정보**: "달라지는 외식업 정책 모음"

#### 2. 커뮤니티 콘텐츠

- **사장님 스토리**: 성공 사례, 일상
- **리뷰/후기**: 파트너어워즈 수상자

#### 3. 지원 콘텐츠

- **정부지원금 안내**
- **프랜차이즈 브랜드 소개**
- **우대혜택 정보**

### 5.2 콘텐츠 큐레이션 원칙

#### 시의성 (Timeliness)

- "1일 전", "2일 전" 등 게시 시점 명시
- 실시간 기준 인기 콘텐츠

#### 실용성 (Practicality)

- 바로 적용 가능한 팁
- 구체적 수치 제시 (예: "2,700만원 지원")

#### 접근성 (Accessibility)

- 짧은 영상 (1~2분)
- 요약형 제목
- 이모지로 가독성 향상

### 5.3 콘텐츠 톤앤매너

- **친근함**: "사장님", "~해요" 반말/존댓말 혼용
- **공감**: "부가세가 너무 많이 나와요ㅠ"
- **격려**: "똑똑하게 장사하는 방법"
- **명확함**: 질문-답변 구조

- 입점 → 운영 시작 → 가게 성장
- 각 단계별 맞춤 지원 제공

## 8. 등대지기 서비스 적용 방안

### 8.1 직접 적용 가능한 요소

#### 1. 정보 아키텍처

```
등대지기 서비스
├── 반려해변입양
│   ├── 커뮤니티 (참여 사례, Q&A)
│   └── 자료실 (가이드, 정화 키트)
├── 해봄 프로그램
│   ├── 커뮤니티 (활동 후기)
│   └── 자료실 (교육 자료)
└── 고객 상담
    ├── FAQ
    └── 전문가 상담 신청
```

#### 2. 콘텐츠 전략

- **트렌드 리포트** → "해양환경 이슈 리포트"
- **장사 캘린더** → "정화 활동 캘린더"
- **성공 사례** → "반려해변입양 수혜 후기"

#### 3. 참여형 콘텐츠

- **퀴즈**: 해양 생태계 상식 퀴즈
- **투표**: 다음 정화 활동 지역 투표
- **Q&A**: 환경 전문가 질의응답

### 8.2 UI/UX 벤치마킹

### 8.3 기능 우선순위

#### Phase 1 (MVP)

1. ✅ 기본 정보 페이지 (반려해변입양, 해봄)
2. ✅ 커뮤니티 게시판 (후기, Q&A)
3. ✅ 자료실 (다운로드)
4. ✅ 문의하기 폼

#### Phase 2 (성장)

1. 전문가 Q&A 시스템
2. 활동 캘린더
3. 퀴즈/이벤트 시스템
4. 숏폼 비디오 콘텐츠

#### Phase 3 (확장)

1. 사용자 대시보드 (참여 이력)
2. 포인트/뱃지 시스템
3. 소셜 공유 기능
4. 실시간 알림

### 8.4 콘텐츠 제작 가이드

#### 제목 작성 원칙

- **문제 해결형**: "해변 쓰레기, 어떻게 처리하나요?"
- **혜택 강조형**: "반려해변입양 신청하고 정화키트 무료로!"
- **공감형**: "처음 참여했는데, 이렇게 감동적일 줄이야"

#### 본문 구조

1. **Hook** (1문단): 공감/문제 제기
2. **Body** (2~3문단): 구체적 정보/가이드
3. **CTA** (마지막): 행동 유도

#### 비주얼 요소

- **Before/After**: 정화 활동 성과
- **인포그래픽**: 환경 데이터 시각화
- **사진**: 참여자 활동 사진 (동의 필수)

### 8.5 측정 지표 (KPI)

#### 참여 지표

- 월간 순 방문자 수 (UV)
- 평균 세션 시간
- 페이지뷰 per 세션

#### 전환 지표

- 반려해변입양 신청 수
- 해봄 프로그램 신청 수
- 문의 접수 건수
- 자료실 다운로드 수

#### 콘텐츠 지표

- 커뮤니티 게시물 수
- 댓글/반응 수
- 콘텐츠 공유 수

## 9. 핵심 인사이트 요약

### 9.1 배민 CEO의 성공 요인

1. **종합 플랫폼 전략**
   - 단순 정보 제공을 넘어 교육, 커머스, 커뮤니티 통합

2. **사용자 중심 설계**
   - 사장님의 실제 고민 해결 (부가세, 인건비 등)
   - 즉시 사용 가능한 도구 제공

3. **콘텐츠 다양성**
   - 텍스트, 영상, 퀴즈, Q&A 등 다채로운 포맷

4. **지속적 참여 유도**
   - 실시간 업데이트
   - 이벤트/혜택 지속 제공
